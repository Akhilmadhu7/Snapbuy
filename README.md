# Snapbuy (E-commerce platform for apple products)
<br/>
SnapBuy is a Ecommerce Web Application for apple products.Implemented all the necessary E-commerce functionalities

# Live App
<br/>
Hey, check out the Live-App, just visit https://snapbuy.website/

Features 
<br/>
* Developed interactive user interface with responsive design
* Custom admin panel with sales report, chart & graph product and order management
* Wishlist,cart and search products
* Offers and coupons
* API integration of payment gateways of Paypal and Razorpay
* Product, user & category management on admin side
